Our Course

Zero Knowledge Course for Developers

Designed for developers who want to get into Zero Knowledge development. This course does NOT go into ZK mathematics or theory, since most of that is not needed for ZK development.

Course Contents

  • An interactive, introductory lecture on Zero Knowledge
  • An interactive lecture on Zero Knowledge development
  • One private session
  • A hands-on end project to solidify your knowledge
  • Access to a private Discord channel for participants

What you'll learn

After completing the course, you will:
  • Understand what is ZK
  • Understand some of the use cases
  • Have experience in programming ZK applications
  • Be part of a supportive ZK community

Prerequisites

There are no hard prerequisites for the course. However, since this is a developer course, you get a lot more out of this if you already know some development.

If you wish to develop the end project, which is highly recommended, you should at least be comfortable with:
  • Writing code in some modern language
  • Finding solutions to your programming issues online
  • Using the terminal
