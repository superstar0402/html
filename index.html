Our Course

Zero Knowledge Course for Developers

Designed for developers who want to get into Zero Knowledge development.

Course Contents

  • An interactive, introductory lecture on Zero Knowledge
  • An interactive lecture on Zero Knowledge development
  • One private session
  • Exclusive access to a private Discord channel for participants
  • A hands-on end project to solidify your knowledge

Next course cohort

The next course will start around mid-October 2023. The first course cohort will accept only four participants.

The lectures will be held somewhere between 10am and 8pm GMT. The exact timing will be agreed upon with the participants.

Prerequisites

There are no hard prerequisites for the course. However, since this is a developer course, you get a lot more out of this if you already know some development.

If you wish to develop the end project, which is highly recommended, you should at least be comfortable with:

  • Writing code in some modern language
  • Finding solutions to your programming issues online
  • Using the terminal

Pricing

The course is priced at 400€ + 21% VAT. However, if you apply through a company, VAT is not applicable.

The payment can be made via cryptocurrencies (for individuals) or euros.
